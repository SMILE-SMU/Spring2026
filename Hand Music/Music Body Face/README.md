# Project: Hand Music

## Purpose
A web app that will be a musical instrument for a specific work of music for the SMILE - Southern Methodist Interdisciplinary Live Ensemble. Movement drives musical events.

## Tech
- TypeScript
- SvelteKit
- Web Audio API
- Mediapipe
- No external math libraries unless necessary

## Constraints
- Must run client-side
- No server dependency
- Prefer explicit math over black-box helpers

## Style
- Clear, minimal
- Avoid over-engineering
